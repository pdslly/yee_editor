# yee-editor    

![yee-editor](https://i.ibb.co/TTHfZsH/yee.png)

[预览地址](https://ifront.net/yee/)  
简单的可视化在线编辑器，帮助快速生成包含丰富动画的H5页面

## V1.1更新内容
- 新增特效组件 (src/widgets/effect)
- 新增全局配置组件 (src/widgets/config) 音乐和脚本组件设置为了全局唯一
- 修复了一些已知问题

## Project setup
```
npm install
```
